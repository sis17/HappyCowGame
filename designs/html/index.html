Happy Cow Game The HTML Mock-Ups

Event Phase

The Event Phase shows the game in the first phase. The event at the beginning of the round is shown in the main screen.

Cards Phase

The Cards Phase shows the game in the second phase. The player's cards are shown, as well as existing rations. There is the chance for a user to create a new ration, if he has enough cards.

Movement Phase

The Movement Phase shows the game in the third phase. The player is shown his rations from which he choses 1, then the dice are calculated and shown, again he chooses 1, then he can move his ration shown on the game board in the middle of the screen.

Review Phase

The Review Phase shows the game in the fourth phase. The user can see a record of what has happened this round, and when satisfied with looking through it, can end their turn.
